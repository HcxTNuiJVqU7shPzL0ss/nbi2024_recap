
# Veckouppgift 1

---

## Vecka 3, 12/1

(_2024 == Vecka 2, 8/1_)

---

1. Du ska lösa uppgiften självständigt eller tillsammans 
med en annan student. Men vid minst ett tillfälle ska du 
göra code review i en grupp.  
   1. Poängen med code review är att man lär sig mycket 
      bättre genom att både skriva och prata kod.
<BR><BR>
2. För att lösa uppgifterna behöver du **PyCharm** 
(eller VS Code) och ett **GitHub**-konto.
<BR><BR>
3. Du får ta hjälp av AI för att _förklara koncept och 
lösa fel_. Du får inte be AI _lösa uppgiften_ åt dig direkt. 
Om du gör det, kommer du inte att lära dig grunderna, 
och inte kunna lösa svårare problem.
<BR><BR>
4. Uppgifterna i den här kursen lämnas in via länk i veckoschemat.

### Assignment Information

#### 1a

Använd input för att be användaren om ett heltal.  
Spara värdet i en variabel.  
Omvandla variabelns värde till ett heltal, och skriv ut det för att 
testa om du har gjort rätt.  

Kodexempel med input:  

    x = input("Fråga här")

<BR>

Use input to ask the user for an integer.  
Save the value in a variable.  
Convert the value of the variable to an integer.  
Print the value to check it has been done correctly.  

---

#### 1b


Fråga användaren efter ett annat heltal.  
Skriv ut summan av talen, alltså tal1 + tal2.  
Testa genom att hitta på två tal och räkna ut summan i huvudet.  
Kontrollera om programmet räknar rätt.

<BR>

Ask the user for another integer.  
Print out the sum of the integers, i.e. first + second.  
Test by coming up with two numbers and sum it up in your head.  
Check that the program calculates correctly.

#### 2a

Nu är det dags att köpa vinterkläder.  
Du ser en fin jacka som kostar 2000 kronor.  
Jackan är på rea och kostar 75% av originalpriset.  
Skriv ett program som räknar ut hur mycket du behöver betala.  

Använd variabeln:  

    rea_procent = 75.0

Tips: räkna ut rabatten med formeln:  

    slut_pris = pris * rea_procent / 100  

_Note: The formula above is incorrect, should be:

    end_price = price - (price * rea_procent / 100)  

<BR>

Now it is time to purchase winter clothing.  
You spot a new jacket that costs 2000 SEK.  
It is on sale, with a 50% discount.  
Write a program which calculates what you should pay.  

_Note: Differs from -24, where 50% was used._

---

#### 2b

Gör om programmet så att användaren kan skriva in en procentsats.  
Testa genom att hitta på en procentsats och räkna ut vad 
programmet ska svara med, innan du kör det.  
Till exempel 10%, som är 200 kr.  
Då ska jackan kosta 2000 - 200 == 1800 kr.

<BR>

Remake the program to allow the user to input a percentage 
for discount.  
Test by making up a percentage and calculate what the program 
should return, prior to running it.  
E.g., 10%, which is 200 SEK.  
Then the jacket should cost 2000 - 200 = 1800 SEK.
